# 
# AmitCompressor VYO v1.0   
# 

Professional Video Compressor for Termux

---

## Installation

```bash
pkg update -y
pkg install git -y

git clone https://github.com/amitdaskne/amitcomp.git

cd amitcomp

bash install.sh
```

---

## Anytime You Want To Open AmitCompressor

```bash
amitcomp
```

---

## Direct Python Run

```bash
python amitcomp.py
```

---

## Uninstall

```bash
bash uninstall.sh
```

---

## Features

- Fast Compression
- FFmpeg Based
- Beautiful CLI UI
- Progress Bar
- Low / Medium / High Compression
